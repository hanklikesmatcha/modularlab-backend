# This is a project for the ModularLab's backend

# Pre-requisite
1. Install Docker

2. Prepare ENVIRONMENT VARIABLES
```
DATABASE_URL=
HUGGING_FACE_TOKEN=
HUGGING_FACE_API=
AZURE_COGNITIVE_SERVICES_KEY=
```

# Start
1. Build containers
```
    docker-compose build
```
2. Spin up the containers
 ```
    docker-compose up
 ```
